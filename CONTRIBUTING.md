# Contributing

Thanks for helping improve VeilVote.

1. Fork the repository and create a focused branch.
2. Keep contract changes small and document privacy assumptions.
3. Add or update tests for every behavior change.
4. Run `sforge fmt --check` and `sforge test -vv` from `contracts/`.
5. Open a pull request explaining the problem, approach and security impact.

Please report vulnerabilities privately as described in [SECURITY.md](SECURITY.md).
